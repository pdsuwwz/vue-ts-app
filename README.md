## 使用技术：

> Vue2 + VueRouter3 + Vuex3 + Webpack4 + Babel7 + express


## 开发文档：

```
// 安装依赖
npm install OR yarn install
```

---

```
// 测试环境
npm run dev

// 生产环境
npm run build

OR

// 直接使用 WDS
npm start

```
